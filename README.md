# МЕХАНИК.РФ: каталоги

Статический сайт с двумя каталогами:

- `Станки`
- `Расходные материалы`

## Как обновить PDF-каталоги

Обновлять нужно только два файла в папке `docs` на GitHub:

- `docs/machines.pdf` — каталог станков
- `docs/materials.pdf` — каталог доп. оборудования / расходных материалов

Важно: новые файлы должны называться точно так же.

## Что происходит автоматически

После замены PDF GitHub Actions запускает пересборку:

1. Считает количество страниц в новом PDF.
2. Нарезает PDF на изображения страниц.
3. Обновляет счетчик страниц в `site.js`.
4. Сохраняет готовые страницы обратно в репозиторий.

Ссылка на скачивание PDF не меняется, потому что сайт всегда ведет на те же файлы:

- `./docs/machines.pdf`
- `./docs/materials.pdf`

## Как проверить обновление

1. Откройте вкладку `Actions` в репозитории.
2. Найдите workflow `Update catalog pages from PDFs`.
3. Дождитесь зеленой галочки.
4. Откройте сайт и обновите страницу в браузере.

Если браузер показывает старую версию, обновите страницу с очисткой кеша или подождите несколько минут.
